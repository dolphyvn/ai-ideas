# AI Venture Spec

## Name:
**HandTheraPro - CHT Documentation & Workers' Comp Billing**

## Core Concept:
Specialized EMR/documentation platform for Certified Hand Therapists (CHT) focusing on workers' compensation billing, outcome tracking, and return-to-work documentation.

## Problem:
- **Only 7,000 CHTs** nationwide (highly constrained specialist pool)
- **Workers' comp complexity**: Primary revenue source with extensive documentation
- **Manual outcome tracking**: DASH, QuickDASH scores measured manually
- **Fabrication billing**: Custom splints require detailed time/materials documentation
- **General PT platforms**: Lack hand-specific assessment tools (grip/pinch strength, ROM)

## Target Vertical:
**Primary:** Certified Hand Therapists (CHTs)
- Private practice owners (have purchasing autonomy)
- Hand therapy practices (2-5 therapists)
- Physical therapy practices with hand therapists on staff

**Secondary:** Occupational medicine practices, hand surgery groups, workers' comp insurers

## Why Now:
1. **Workers' comp scrutiny increasing**: Documentation requirements tightening
2. **Aging workforce**: Hand injuries common in aging population
3. **Post-surgical rehab**: Complex procedures (joint replacements, fractures) increase
4. **Technology adoption**: Digital measurement tools becoming standard
5. **CHT credential growth**: More therapists specializing in hand therapy

## AI Advantage:
- **Hand-specific assessments**: Grip/pinch strength, ROM measurements, edema tracking
- **Outcome measures**: DASH, QuickDASH scoring automation
- **Workers' comp billing**: Treatment plan updates, return-to-work documentation
- **Fabrication billing**: Custom splint documentation (materials, time, labor)
- **Referral coordination**: Hand surgeons, physicians, therapists

## Viral Mechanism:
- **ASHT conferences**: American Society of Hand Therapy annual meeting
- **Specialist networks**: CHT referral communities
- **Case studies**: "Reduced return-to-work time by 40%"
- **Hand surgeon partnerships**: Orthopedic groups refer patients

## Revenue Model:
**Per-Therapist Subscription:**
- **Solo**: $150/month - Single CHT
- **Practice**: $400/month - Up to 5 therapists
- **Clinic**: $800/month - Unlimited + workers' comp billing module

**Per-Case add-on:**
- $50 per workers' comp case (billing documentation)

**Implementation:**
- $2,000 one-time (assessment tools, integration, training)

**Unit Economics:**
- CAC: $80 (ASHT conference, specialist networks)
- LTV: $4,320 (36-month retention)
- Gross Margin: 80%

## Moat:
1. **Credential moat**: CHT certification (4,000+ hours, rigorous exam)
2. **Small specialist pool**: 7,000 CHTs nationally
3. **Workers' comp complexity**: Billing, documentation, return-to-work tracking
4. **Niche market**: Too small for general PT EMRs to care
5. **Specialized assessments**: Hand-specific measurements not in general platforms

## MVP in 12 Weeks:
**Weeks 1-3:**
- Focus: Workers' compensation billing only
- Manual CHT documentation workflow
- Basic outcome tracking (DASH)

**Weeks 4-6:**
- Build AI documentation generator (treatment notes, progress)
- Workers' comp billing module (treatment plan updates)
- Outcome measurement automation

**Weeks 7-9:**
- Partner with 2 hand therapy practices
- Deploy pilot
- Validate: Billing acceptance, documentation time reduction

**Weeks 10-12:**
- Add hand assessment tools (grip/pinch strength, ROM)
- Fabrication billing (custom splints)
- Prepare ASHT conference presentation

**Validation Metrics:**
- 50%+ documentation time reduction
- 95%+ workers' comp claim acceptance
- 2+ practices commit after pilot
- Outcome tracking measurable

## Tech Stack:
- **Backend**: Python/FastAPI
- **AI**: GPT-4 (documentation), Custom ML (outcome prediction)
- **Integrations**: Workers' comp payers, hand surgeon referral systems
- **Infrastructure**: AWS (US regions)

## Monthly Revenue Potential (Year 1–3):
- **Year 1**: $15K MRR (150 CHTs × $100 ARPU)
- **Year 2**: $75K MRR (750 CHTs)
- **Year 3**: $200K MRR (2,000 CHTs + clinic licenses)

## Risk Factors:
1. **Small market**: Only 7,000 CHTs nationally
2. **Competition risk**: WebPT, TheraBill could add hand features
3. **Adoption resistance**: Some CHTs prefer paper/general platforms
4. **Workers' comp complexity**: Payers have specific requirements
5. **Credential requirement**: Need CHT input for product development

## Competitive Threat:
- **WebPT**: General PT EMR, lacks hand-specific tools
- **TheraBill**: PT billing, not workers' comp focused
- **Clinicast**: PT documentation, not hand-specific
- **General PT platforms**: Lack specialized hand assessments

**Differentiation**: Only platform designed specifically for hand therapy with workers' compensation billing, hand assessment tools, and CHT-level documentation requirements.
