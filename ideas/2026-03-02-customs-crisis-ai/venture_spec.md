**Date:** 2026-03-02

## Core Concept

AI-powered CBP/FDA/USDA import detention defense platform. Predicts detention risk from species/country hazard patterns, automates compliance documentation, and provides crisis response when shipments are detained at the border. "Permit-blocking at the border" - cargo cannot clear without compliance.

**The Bridge:** Reactive Import Defense (detention = panic, total cargo loss) → [AI] - Predictive Risk Monitoring (detention prevention, crisis preparedness)

---

## Problem Statement

### The Import Detention Crisis

- **150K+ importers** face growing detention risk
- **Detention = total cargo loss** (perishables destroyed, supply chain disrupted)
- **$50K-500K per detention** (cargo value + demurrage + lost sales)
- **FDA/USDA/CBP coordination complex** (3 agencies, different rules)
- **Species/country hazard patterns** predictable from OAI refusal data

### The Current Reality

```
"We had a shipment of organic supplements detained at LA port for 3
weeks. FDA questioned FSVP documentation. By the time we responded,
$120K of product was expired. We lost the entire shipment. If we'd
had warning that this SKU from Vietnam was high-risk, we would have
prepared documentation in advance. We'd pay $15K immediately for
a service that prevented detentions."
- Import Manager, Supplement Distributor
```

## Why Now

### 1. FDA Import Enforcement Surge

- FDA FSVP enforcement #1 priority (import food safety)
- Import detentions up 60% since 2021
- OAI (OAI Import Refusal Reports) publicly available

### 2. Supply Chain Disruption Economics

- Detention = supply chain shutdown (existential to business)
- Perishable products destroyed during detention
- Demurrage fees accumulate daily
- Just-in-time inventory means no buffer

### 3. Multi-Agency Complexity

- FDA (food safety), USDA (animal/plant), CBP (tariffs)
- Each agency has different documentation requirements
- Coordination complexity creates detention risk

### 4. Data Moat Available

- FDA OAI refusal data = species/country hazard patterns
- CBP ACE/OAI data = enforcement trends
- ML can predict which SKUs from which countries = high risk
