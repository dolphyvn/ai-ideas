## Core Concept

AI-powered claim file documentation automation for independent insurance adjusters. The system auto-generates complete claim file documentation from adjuster notes and evidence, using templates derived from public claim file subpoenas and discovery requests (showing what courts consider "adequate" documentation).

**The Bridge:** Manual Claim Documentation (10-20 hours per claim, 30-50% of time) → [AI] → Automated Documentation Generation (notes → complete file in minutes)

---

## Problem Statement

### The Claims Documentation Burden

- **500K+ independent adjusters** in the US
- **30-50% of claim time** spent on documentation (non-billable)
- **10-20 hours per claim** on contact logs, investigation notes, coverage analysis
- **Adjusters bill $50-150/hour** - documentation is pure overhead
- **Claim litigation subpoenas** reveal documentation gaps (bad faith risk)

### The Current Reality

```
"I handle 80 claims a year. I spend at least 15 hours per claim just
documenting files. That's 1,200 hours annually of work I can't bill.
When I'm documenting, I'm not adjusting new claims. I'd pay $200/month
for AI that reads my messy notes and generates court-ready claim files."
- Independent Property Adjuster, Catastrophe Team
```

## Why Now

### 1. Public Litigation Data Available

- PACER provides court filings, discovery requests, subpoenas
- Claim file litigation reveals "adequate" documentation standards
- Real-world examples provide training data

### 2. Underserved Market

- Carrier-focused incumbents (Guidewire, Snapscreen) serve large carriers
- Independent adjusters handle 60-80% of claims
- Price-sensitive but high-volume market

### 3. Documentation Pain Universal

- 30-50% of claim time is documentation (hated by all adjusters)
- Voice-to-text, mobile notes create messy input data
- LLMs excel at structuring unstructured notes

### 4. Viral Community

- Adjuster forums (IA Adjuster, Facebook groups) highly active
- Productivity tips shared enthusiastically
- "Saved 10 hours per claim" is powerful testimony
